# 🌱 Paradise Nursery Shopping App

This is a React-based shopping cart application for an online plant store called **Paradise Nursery**.

## 🚀 Features

* Browse different plants
* View plant details (name, price)
* Add plants to cart
* Increase/decrease quantity
* Remove items from cart
* Dynamic total price calculation

## 🛠️ Tech Stack

* React.js
* JavaScript
* HTML & CSS

## 📂 Project Structure

* App.js
* Navbar.js
* ProductList.js
* Cart.js
